# bright-misery
A brightness controller for Intel laptops using Linux

### INSTALL

Simply run install.sh.

```shell
$ cd bright-misery
$ chmod +x install.sh
$ ./install.sh
```

### USAGE

```shell
$ bm [val]
```

val: signed integer from -100 to 100 rappresenting the percentage you want to add/remove.

__use at your own risk__
